        Activity 1
        1)
        P(2): Stealing physical property is morally wrong.
        P(1): Downloading proprietary software (without permission) is identical to stealing physical property.
        C(0) Downloading proprietary software (without permission) is morally wrong.
        2) The argument is valid I've no counterexample, and the colour-coding shows how the conclusion follows from the premises; "stealing physical property" is the middle term.
        3) Yes, the argument is valid.
        4) The argument is unsound in the actual world, downloading proprietary software (without permission) is not identical to stealing physical property; downloading software does not remove possession of the software from any of the owners.
        6) No, at least one of the premises is false in the actual world.
        7) The argument is weak it is valid, but unsound (#2).
         
        Example 1
        1)
        P(2): The redwood is a tree.
        P(1): All trees are plants.
        C(0) Redwood is a plant.
        2) The argument is valid I've no counterexample, and the colour-coding shows how the conclusion follows from the premises; "tree" is the middle term.
        3) Yes, the argument is valid.
        4) The argument is sound all of the premises are true in the actual world.

        6) Yes, all of the premises are true in the actual world.

        7) The argument is strong it is sound (#1).
         
        Example 2
        1)
        P(2): The Earth is round.
        P(1): All basketballs are round.
        C(0) The Earth is a basketball.
        2) The argument is invalid the conclusion doesn't follow from the premises; two "-thing"s being round, does not mean they are the same "-thing".
        3) No, the argument is invalid.
        5) The argument is fallacious the Earth is a beachball, which is very different from a basketball.
        6) Yes, all of the premises are true in the actual world.
        7) The argument is weak it is fallacious with some mixture of true and false premises (#5).

        PSSSS: There're other techniques for (dis)proving arguments.
         
        Argument Structure

        P = P(1) P(2) P(3)

        Q = C(0)
        P Q
         
        Direct Proofs
        Assume P is true
        Deduce Q is true
        P Q
         
        Contrapositive Proofs
        Assume Q is false
        Deduce P is false
        (P Q) = (Q P)
         
        Contradictory Proofs
        Assume P is true
        Assume Q is false
        Disprove P Q
        (P Q) = (P Q)
         
        Trivial Proofs
        Trivially true when Q(x)
        x: P(x) Q(x)
         
        Vacuous Proofs
        Vacuously true when P(x)
        x: P(x) Q(x)
         
        Counterexample Disproofs
        Prove P
        P P
        :  T F = F
         
        Inductive Proofs
        (P(1) (k: P(k) P(k + 1))) (n: P(n)), where k 1 n
